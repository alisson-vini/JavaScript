Formas de usar o JavaScript dentro do seu site

Interno

Existem duas formas de usar JavaScript dentro do seu site, a primeira é coloca-lo dentro do próprio arquivo HTML usando "Script".

Externo

A outra forma é a mais comum e mais organizada, deixando um arquivo separado somente para o JavaScript quando fazemos dessa forma é uma boa prática deixar o Script no final do documento HTML pois o navegador deve processar o JavaScript por ultimo


Saídas para os Scripts JS

Alerta - confimação - input

Um dos tipos de saídas são mensagens na tela, podemos fazer isso com:

alert exibe uma mensagem de Alerta

confirm exibe uma mensagem de confirmação

prompt permite mandar um input como um nome ou idade
